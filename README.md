# 🏗️ Caucasia Heavy-Import Hub
### Ecosistema Digital de Importación de Maquinaria Pesada

**Desarrollado por:** Vibras Positivas HM  
**Ubicación:** Caucasia, Antioquia, Colombia  
**Versión:** 1.0.0 (Abril 2026)

---

## 📋 Descripción
Esta es una **PWA (Progressive Web App)** diseñada específicamente para empresarios mineros y agrícolas del Bajo Cauca Antioqueño. Permite gestionar el proceso completo de importación de maquinaria pesada desde China a Colombia, integrando normativas legales vigentes a 2026.

## 🚀 Características Principales
- **Liquidador DIAN 2026:** Cálculos automáticos de Arancel e IVA basados en el **Decreto 0264** (Siderúrgicos).
- **Catálogo Minero:** Comparativa en tiempo real entre el costo de importación FOB y el precio de venta en el mercado colombiano.
- **Negociación Directa:** Generador de plantillas de correo técnico en inglés para fábricas chinas (SANY, XCMG, etc.).
- **Funcionamiento Offline:** Gracias al Service Worker, la app funciona en zonas rurales con baja conectividad.
- **Optimización Logística:** Enfoque prioritario en el arribo por **Puerto Antioquia (Urabá)**.

## 🛠️ Tecnologías Utilizadas
- **Frontend:** HTML5, CSS3, JavaScript (ES6+).
- **PWA:** Web App Manifest y Service Workers para instalación nativa.
- **Legal Engine:** Lógica programada con la normativa aduanera colombiana de abril 2026.
